# Portfólio — Gabriel Navarro

Stack: React + Vite (ESM), CSS puro (dark minimalista).

## Rodar localmente
```bash
cd frontend
npm install
npm run dev
# abra http://localhost:5173
```
- Currículo: `#/curriculo` → http://localhost:5173/#/curriculo
- Baixar PDF: botão na Navbar ou /CV_Gabriel.pdf
